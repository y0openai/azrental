## 🎯 1. 톤 & 매너 (Voice & Tone)

### 1.1 핵심 가치

| 톤 | 설명 | 예시 |
|---|---|---|
| **따뜻함** | 함께한다는 느낌 | "함께 시작해요", "당신을 응원해요" |
| **신뢰** | 명확하고 솔직하게 | "100% 신원 확인됨", "안전해요" |
| **전문성** | 실용적이고 효율적으로 | "3단계 예약", "즉시 확인" |

### 1.2 말투 원칙

**존댓글 사용하기 (정중함)**
```
✅ "예약되었어요"
✅ "완료했어요"
✅ "알려드렸어요"
❌ "예약되었습니다" (너무 딱딱함)
❌ "예약됐어" (너무 캐주얼함)
```

**짧고 명확하게**
```
✅ "아직 신청 중이에요"
✅ "미리 취소하면 환불이 더 많아요"
❌ "현재 검증 프로세스가 진행 중인 상태입니다"
❌ "24시간 이전 취소 시 환불 수수료가 공제되지 않습니다"
```

**문제보다 해결책**
```
✅ "다른 날짜를 선택해보세요"
✅ "비밀번호를 다시 입력해주세요"
❌ "예약할 수 없는 날짜입니다"
❌ "비밀번호 오류입니다"
```

**긍정적으로 표현**
```
✅ "환불 가능해요"
✅ "더 자주 만나도 좋아요"
❌ "환불 불가능하지 않습니다"
❌ "제한이 없습니다"
```

### 1.3 피해야 할 표현

❌ "아재들 렌탈", "오빠들 픽업" (너무 캐주얼)
❌ "노인 서비스", "할머니 할아버지" (존중 부족)
❌ "우리 서비스는 최고입니다" (과장)
❌ "본 서비스는..." (딱딱함)

## 🎬 2. 랜딩 페이지 콘텐츠 (재구성 5섹션)

### 2.0 Section 0: Hero (서비스 정의 + 캐주얼한 제안)

**H1 (재치있는 아저씨 톤)**
```
혼자하긴... 좀 외로운데?
```

**부제 (공감과 제안)**
```
누군가의 경험이 필요하신가요?
그럼 우리 아조씨들을 만나봐요.
```

**이중 CTA** (명확한 행동 유도)
```
🔍 아조씨 찾기      (파란색 - 고객)
🎯 아조씨 되기      (초록색 - 공급자)
```

**창업자 스토리 모달 버튼**
```
💭 창업자의 생각이 궁금하신가요?
```

**역할**:
- 뭔가? (서비스 정의 + 매력적인 예시)
- 왜? (따뜻함과 신뢰)
- 어떻게? (이중 CTA)
- 추가: 창업자 스토리는 모달/토글로 선택적 공개
- 시간: 3-5초

---

### 2.0-1 Section 0.5: 아조씨와 함께 할 수 있는 일들 (캐주얼 서비스 가이드)

**섹션 제목**
```
아조씨와 함께 할 수 있는 일들
```

**섹션 설명 (재치있는 톤)**
```
조언쟁이? 아니에요. 그냥... 당신의 일상에 함께 있고 싶은 아저씨들이에요.
```

**12가지 캐주얼한 서비스 카드 (재치있는 톤)**

| # | 서비스 | 설명 (아저씨 톤) | 이모지 |
|---|--------|------|--------|
| 1 | 잡담 상대 | 누군가와 그냥... 떠들고 싶은 날 있잖아. 그럴 땐 우리를 불러봐 | 💬 |
| 2 | 조언/어드바이스 | "이거 어떻게 할까?"라고 물어볼 누군가가 필요할 때 | 💡 |
| 3 | 음식점 동행 | 그 식당, 혼자 가긴 좀 어색한데... 우리랑 같이 가볼까? | 🍽️ |
| 4 | 취미 함께하기 | 영화도 좋지만, 함께 보면 더 얘기 많아지더라고요 | 🎮 |
| 5 | 스포츠 관전 | 경기장에서 함께 응원하면 재미가 배로 느껴져요 | ⚽ |
| 6 | 전직/커리어 상담 | 새로 시작하려니까 불안한가요? 경험자의 이야기를 들어보세요 | 💼 |
| 7 | 진로/자기개발 | 미래가 막연할 땐, 아저씨들의 인생 경험이 나침반이 되어줄 수 있어요 | 🎓 |
| 8 | 방 정리/정돈 | 혼자 하면 지치는 일, 함께하면 수다도 나누고 팁도 배워요 | 🏠 |
| 9 | 캠프/야외활동 | 산에 가봤자 혼자면 외로운데, 우리가 함께 즐겨봐요 | 🏕️ |
| 10 | 논문/글 첨삭 | "이 문장 좀 봐봐" 할 때, 경험 많은 눈으로 봐드려요 | 📝 |
| 11 | 드라이브 | 드라이브하다 보니까 말도 나오고 경치도 나오더라고요 | 🚗 |
| 12 | 라이브/공연 동행 | 좋아하는 가수의 공연, 함께 가면 그 감동이 두 배예요 | 🎸 |

**추가 설명 (재치있는 톤)**
```
✨ 이 정도면 충분하지 않나요? 다른 것도 있으면 얘기해 봐요. 뭐든 함께할 수 있어요.
```

**역할**:
- 서비스의 다양성과 캐주얼함 강조
- "거창한 서비스"가 아닌 "생활 밀착형" 감성 전달
- 누군가는 제공자, 누군가는 고객이 될 수 있음을 암시

### 2.1 Section 1: How It Works (3단계 프로세스)

**섹션 제목**
```
어떻게 시작하나요?
```

**3단계 (명확하고 간결, 재치있는 톤)**

**Step 1. 아조씨 찾기**
```
필요한 분야, 경험, 취향...
당신에게 맞는 아조씨를 찾아봐요.
```

**Step 2. 예약하기**
```
언제, 어디서, 뭘 할지 정하고
예약하면 끝.
```

**Step 3. 만나서 대화**
```
카페, 영상통화, 산책...
편한 곳에서 시간을 함께 써요.
```

**역할**:
- 심플함 전달 ("3단계면 돼")
- 사용자 안심 ("복잡하지 않아요")

### 2.1-1 Section 1.5: 아조씨들을 만나보세요

**섹션 제목**
```
실제로 활동 중인 아조씨들
```

**섹션 설명**
다양한 경력과 전문성을 가진 아조씨들을 소개합니다. 각자의 개성 있는 경험과 이야기를 직접 만나보세요.

- 심플하면서도 개성 있는 소개로 사용자 공감 형성

#### 2.2.3 안전 정책 (재치있는 톤으로 불안감 해소)

**제목 (재치있는 톤)**
```
🔒 물론, 안전이 첫 번째겠지
```

**4가지 안전 장치 (대화체)**
```
✓ 누구인지 확인하고 시작
✓ 돈은 안전하게 에스크로우로
✓ 뭔가 문제? 24시간 우리가 있어
✓ 마음 안 들면 환불 OK (하루 전)
```

**경계선 명시** (신뢰 구축)
```
"아조씨는 친구도, 연인도 아니에요.
믿을 수 있는 이웃입니다."

할 수 있어요        할 수 없어요
─────────────────────────────
대화하기            신체적 접촉
카페/식사           연애/데이팅
조언 구하기         성인 서비스
감정 나누기         불법 활동
활동 함께하기       강압이나 폭력
```

**역할**: 마지막 불안감 해소

### 2.3 Section 3: Final CTA (재치있는 톤으로 행동 유도)

**제목 (재치있는 제안)**
```
그러면, 시작해 볼까?
```

**리드 문구 (공감 중심)**
```
누군가의 경험이 필요하신가요?

```

**이중 CTA**
```
[아조씨 찾아보기]  (파란색)
[나도 아조씨 하기]  (초록색)
```

**추가 지원 (친근한 톤)**
```
궁금한 게 있어? 자주 묻는 거들을 봐봐.
```

**역할**: 명확한 행동 유도 + 다음 스텝

## 🎭 2.5 모달: 창업자 스토리 (Optional Deep Dive)

**활성화**: Hero 섹션의 "💭 창업자의 생각이 궁금하신가요?" 버튼 클릭 시

**모달 제목**
```
창업자의 생각
```

**모달 콘텐츠 (3개 섹션, 재치있는 톤)**

### 섹션 1: 🧠 깨달음 (문제 인식)
```
요즘 유튜브 댓글을 보면... 중년 남성을 깎아내리는 영상들 천지예요.

'영포티', '아재들', '구닥다리'...

볼 때마다 답답했어요. 이게 뭐 하는 짓인가 싶고.
```

### 섹션 2: ❓ 의문 (감정적 공감)
```
우리가 정말 중년 남성을 이렇게 봐야 하나?

이들도 누군가의 아버지고, 누군가의 인생 선배인데...
경험이랑 지혜가 진짜 쓸모없는 거야?

그럴 리 없지, 라고 생각했어요.
```

### 섹션 3: 💪 결심 (해결책)
```
그래서 시작했어요.

중년 남성들의 경험과 지혜를 부정하지 말고,
그걸 필요로 하는 사람들과 직접 연결하는 거.
그게 아조씨 렌탈이에요.

함께라는 것의 참 의미를 다시 찾고 싶었어요.
```

**모달 배경**: 창업자 인물사진 또는 감성적 이미지
**모달 닫기**: 안 보기 (X 버튼), 기간 설정 가능 (예: 일주일 동안 안 보기)

**역할**:
- 사용자의 선택을 존중 (보고 싶은 사람만 보기)
- 깊이 있는 스토리텔링으로 신뢰와 감정 이입 강화
- 브랜드 가치 전달 (중년 남성 존중, 공동체)

| S1.5 | 프로필 카드 | 3명 샘플 아조씨 (영화/요리/사진) | 구체적 인물로 신뢰 구축 |

| S2 | Trust | 신뢰지표 + 후기 + 안전정책 | 신뢰 통합 |

| MODAL | 창업자 스토리 | 깨달음 → 의문 → 결심 | 선택적 심화 학습 |

**재구성 변경사항 (2026-01-11)**:
- **Hero 개선**: 창업자 스토리를 Hero에서 분리 → 모달로 (사용자 선택권 존중)
- **Section 0.5 신규 추가**: "아조씨와 함께 할 수 있는 일들" (12가지 캐주얼 서비스)
  - 목적: 서비스의 다양성과 일상성 강조
  - 톤: 거창하지 않은 따뜻한 감성
  - 배치: Hero 직후 (즉시 매력 전달)
- **창업자 스토리 모달 추가**:
  - 활성화: Hero 버튼 클릭
  - 구성: 깨달음 → 의문 → 결심 (3단계)
  - 역할: 신뢰와 브랜드 가치 심화
  - UX: "안 보기" 옵션 (존중)

**톤 정제 (2026-01-11)**:
- **Hero**: "혼자하긴... 좀 외로운데?" (공감 중심)
- **Section 0.5**: 12개 서비스 모두 재치있는 아저씨 톤으로 변경
  - 예: "누군가와 그냥... 떠들고 싶은 날 있잖아. 그럴 땐 우리를 불러봐"
- **How It Works**: 각 스텝을 대화체로 변경
  - 예: "언제, 어디서, 뭘 할지 정하고 예약하면 끝."
- **Trust/Reviews**: 리뷰를 진솔한 톤으로 변경
  - 예: "박준호 아조씨 만나고 인생이 좀 달라졌어요."
- **Safety**: 제목과 항목 모두 친근한 톤으로 변경

- **Final CTA**: "그러면, 시작해 볼까?" (제안 톤)
