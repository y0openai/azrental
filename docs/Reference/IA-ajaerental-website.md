# 아재렌탈(AZ help me) 웹사이트 IA (Information Architecture)

**작성자**: 신달수
**작성일**: 2026년 1월
**전략**: 창업자 중심 IA (신달수가 브랜드 얼굴)

---

## 📐 핵심 전략: 창업자 중심 구조

### 왜 창업자가 얼굴이 되는가?

**일본 모델 분석 (Ossan Rental)**:
- Takanobu Nishimoto가 창업자이면서 동시에 첫 번째 "렌탈 아재"
- 창업자 본인의 스토리가 서비스의 핵심 가치
- "왜 이 서비스를 하는가"를 창업자의 철학으로 전달

**한국 적용 (아재렌탈)**:
- 신달수가 단순 CEO가 아닌 **브랜드 대사**
- "영포티 문화 반발" 스토리가 마케팅의 핵심
- 창업자의 정체성 = 서비스의 가치

## 👥 주요 User Flow

### Flow 1: 신규 고객 여정

```
1. 홈 방문
   ↓
2. 신달수의 메시지 읽기
   "왜 영포티 문화가 싫어서 이 사업을 했는가"
   ↓
3. "왜 아재렌탈인가?" 읽기
   ↓
4. 아재 검색/필터
   (지역, 특기, 카테고리)
   ↓
5. 아재 프로필 확인
   - 사진, 소개, 후기, 평점
   ↓
6. 예약
   - 날짜/시간 선택
   - 활동 내용 입력
   - 결제
   ↓
7. 만남
   ↓
8. 후기 작성
   ↓
9. 재예약 (60% 목표)
```

### Flow 2: 신규 아재(공급자) 여정

```
1. 신달수의 "함께 아재가 되자" 영상/글 보기
   ↓
2. 신달수의 철학 공감
   "중년 남성의 가치를 증명하자"
   ↓
3. 아재 신청 양식 작성
   ↓
4. 신원 확인 및 심사 (1/50)
   ↓
5. 프로필 생성
   - 사진, 소개, 특기
   ↓
6. 온보딩 교육
   - 서비스 철칙
   - 안전 프로토콜
   ↓
7. 스케줄 등록
   ↓
8. 첫 예약 받기
   ↓
9. 커뮤니티 활동
```

## 🎨 페이지별 주요 요소

### 1. 홈 (Landing Page)

**목표**: 신달수의 철학 전달 + 즉시 행동 유도

**주요 섹션**:
```
Header
├─ 로고 (AZ help me)
├─ 네비게이션 (찾기, 커뮤니티, 아재 신청)
└─ 로그인 / 회원가입

Hero Section
├─ 배경: 신달수의 대표 사진
├─ 제목: "중년 남성, 이제 멋진 시대다"
├─ 부제: "신달수가 시작한 아재 이미지 혁명"
└─ CTA: "아재 찾기" / "아재가 되기"

신달수의 메시지 섹션
├─ 신달수 프로필 사진 (큼)
├─ 인사말 (1-2단락)
│  "안녕하세요, 신달수입니다.
│   저는 게임 회사 프로듀서였고,
│   인터넷에서 중년 남성이 부정적으로 표현되는
│   '영포티' 문화에 충격을 받았습니다.
│   중년 남성의 가치를 증명하고 싶었습니다."
└─ 영상 링크 (신달수 인터뷰)

"왜 아재렌탈인가?" 섹션
├─ 문제 제시 (영포티 문화)
├─ 해결책 (아재렌탈)
├─ 일본 성공 사례
│  - 8.7/10 평가
│  - 60% 재이용율
│  - 1,500+ 누적 고객
└─ 한국의 기회 (1인 가구 300만명)

아재들의 이야기 섹션
├─ 아재 3-4명 프로필 + 사진
├─ 각 아재의 한 줄 소개
└─ "아재가 되기" CTA

이용 후기 섹션
├─ 상위 5개 후기 (별점 높은 순)
├─ 고객 이름, 사진, 한 줄 평
└─ "모든 후기 보기" 링크

Footer
├─ 회사 정보
├─ SNS 링크
└─ 법률 링크
```

**목표**: 아재를 찾아서 예약하기까지의 경로 최단화

**주요 요소**:
```

├─ 키워드 검색
├─ 필터
│  ├─ 지역 (서울, 경기, 강원 등)
│  ├─ 특기 (면접코칭, 이사, 노래방 등)
│  ├─ 가격대
│  ├─ 평점 (4점 이상 등)
│  └─ 가능 시간
│
└─ 정렬 (인기순, 신규순, 평점순)

아재 카드 리스트
├─ 각 카드 구성:
│  ├─ 아재 사진 (크고 임팩트 있게)
│  ├─ 이름 / 나이 / 지역
│  ├─ 한 줄 소개
│  ├─ 평점 (★★★★★ 4.9/5.0)
│  ├─ 예약 건수 (210건)

│
└─ 무한 스크롤 또는 페이지네이션
```

## 🎯 정보 구조의 핵심 원칙

### 1. 신달수의 철학이 모든 곳에
- 홈페이지 히어로 → 신달수 메시지
- 아재 검색 → "왜 이들이 멋있나" 스토리
- 신달수 영역 → 창업 철학 상세
- 커뮤니티 → 신달수의 메시지 + 아재 이야기

### 2. 신뢰도 구축 우선
- 신원 확인: "1/50의 엄격한 심사"
- 후기와 평점: 매 페이지마다 보이기
- 안전 규칙: 명확하게 제시
- 미디어 보도: 언론 신뢰도 활용

### 3. 콘텐츠 우선 설계
- 텍스트보다 **사진과 영상**
- 신달수의 메시지는 **영상**으로
- 아재 이야기는 **갤러리**로
- 후기는 **사진과 함께**

### 4. 행동 유도 (CTA) 전략
- 모든 섹션마다 CTA 있음
- "아재 찾기" / "아재 신청" 이원화
- 로그인 전: 개별 CTA
- 로그인 후: 예약 완성형 CTA
