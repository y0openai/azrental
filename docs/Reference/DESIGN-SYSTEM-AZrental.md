# 아재렌탈(AZ help me) 디자인 시스템

**작성자**: 신달수
**작성일**: 2026년 1월
**기준**: Simon Sinek의 구성 + David Whyte의 디자인 기술

---

## 🎯 디자인 철학

### 핵심 원칙
```
Simon Sinek의 구성 (Structure)
"왜 → 어떻게 → 무엇" 순서로 신뢰와 철학 전달

+

David Whyte의 디자인 (Aesthetics)
따뜻함, 영상 중심, 명상적 톤, 인간적 감성
```

### 디자인 목표
- 신달수의 철학을 **따뜻하고 진정성 있게** 전달
- 중년 남성의 가치를 **영감을 주는 방식**으로 표현
- 사용자와의 **감정적 연결** 형성
- **신뢰**와 **공감** 동시에 구축

## 🔤 타이포그래피

### 폰트 선택

#### 한글 (Primary)
```
폰트명: Noto Sans CJK KR
무게별:
  - Bold (700): 헤딩 (H1, H2)
  - SemiBold (600): 소제목 (H3, H4)
  - Regular (400): 본문
  - Light (300): 보조 텍스트

특징: 따뜻하고 친근한 고딕체
적용: 모든 한글 콘텐츠
```

#### 영문 (Supporting)
```
폰트명: Inter 또는 DM Sans
무게별:
  - Bold (700): 강조 영문
  - Regular (400): 본문 영문

특징: 현대적이면서도 읽기 쉬운 산세리프
적용: 영문 로고, 몇몇 영문 요소
```

#### 손글씨/캘리그래피 (Accent)
```
폰트명: 신달수의 손글씨 스캔 또는 손글씨 폰트
사용:
  - "영포티를 반박합니다" (히어로)
  - 에세이 제목
  - 특별한 메시지
  - 서명 (신달수)

특징: 인간미, 진정성, 따뜻함
```

### 타이포그래피 스케일

| 레벨 | 용도 | 크기 (모바일/데스크톱) | 무게 | 라인높이 |
|------|------|-----|------|---------|
| **H1** | 페이지 제목 | 32px / 48px | Bold 700 | 1.2 |
| **H2** | 섹션 제목 | 28px / 40px | Bold 700 | 1.3 |
| **H3** | 소제목 | 24px / 32px | SemiBold 600 | 1.4 |
| **H4** | 카드 제목 | 18px / 24px | SemiBold 600 | 1.4 |
| **Body** | 본문 | 16px / 18px | Regular 400 | 1.6 |
| **Small** | 보조 텍스트 | 14px / 16px | Regular 400 | 1.5 |
| **Caption** | 설명 텍스트 | 12px / 14px | Regular 400 | 1.4 |

### 타이포그래피 원칙

1. **모바일-먼저** (Mobile-First)
   - 최소 16px (본문)로 읽기 쉽게
   - 충분한 라인높이 (1.6 이상)

2. **명확한 계층** (Clear Hierarchy)
   - H1 > H2 > H3 시각적 차이 명확
   - 무게와 크기로 우선순위 표현

3. **따뜻한 톤** (Warm Tone)
   - 딱딱한 Helvetica 대신 친근한 Noto Sans
   - 손글씨로 인간미 표현

└─ 최대 너비: 728px

데스크톱 (1024px 이상)
├─ 컬럼: 12 컬럼
├─ 거터: 24px
├─ 마진: 40px (좌우)
└─ 최대 너비: 1200px
```

### 스페이싱 스케일

```
XS: 4px    (작은 요소 간격)
S:  8px    (버튼 안쪽 패딩)
M:  16px   (기본 요소 간격)
L:  24px   (섹션 내부 간격)
XL: 40px   (섹션 간 간격)
XXL: 64px  (주요 섹션 간 간격)
```

**적용**:
- 텍스트와 이미지 사이: L (24px)
- 섹션 사이: XXL (64px) 이상 (David Whyte처럼 여유로운 여백)
- 버튼 패딩: 좌우 M (16px), 상하 S (8px)

### 여백 활용 (Negative Space)

David Whyte 웹사이트의 특징: **많은 여백으로 호흡하는 공간**

```
페이지 너비 대비 여백 비율:
├─ 히어로: 50% 이미지, 50% 텍스트 + 여백
├─ 본문: 60% 콘텐츠, 40% 여백
├─ 카드: 내부 패딩 L (24px) 이상
└─ 섹션: 위아래 XXL (64px) 이상
```

## 🎬 이미지 & 영상

### 이미지 스타일

#### 1. 신달수 프로필 사진
```
스타일: 자연스러운 일상 사진 (스튜디오 사진 X)
배경: 중립적 배경 (실내 또는 자연)
표정: 따뜻하고 진정성 있는 표정
조명: 자연광 (또는 자연스러운 실내광)
크기: 히어로에서 50% 이상의 가시성

특징:
- 형식적이지 않음
- 인간미 있음
- 신뢰감 있음
```

#### 2. 아재 프로필 사진
```
스타일: 다양한 배경에서의 자연스러운 사진
포함 요소:
  - 일상의 모습
  - 약간의 미소
  - 신뢰감 있는 표정
  - 다양한 연령대와 외모
```

#### 3. 커뮤니티 이미지
```
스타일: 다큐멘터리 스타일의 만남 장면
특징:
  - 진정성 있는 상호작용
  - 자연스러운 감정 표현
  - 다양한 장면 (카페, 공원, 실내 등)
```

### 영상 활용

#### 히어로 섹션 영상
```
길이: 10-15초
내용: 신달수의 일상 모습 + 메시지
특징:
  - 자동 재생 (음소거)
  - 천천한 움직임 (명상적)
  - 자연광 강조
  - 텍스트 오버레이 (메시지)

기술:
  - MP4 + WebM 포맷
  - 모바일 최적화 (세로 또는 정사각형)
  - 폴백: 정적 이미지
```

#### 철학 설명 영상
```
길이: 2-3분
내용: 신달수의 인터뷰 또는 짧은 강연
스타일: David Whyte처럼 명상적이고 깊이 있게
특징:
  - 단순한 배경
  - 자연광
  - 느린 페이스
  - 자막 (자동 재생 시)
```

## ✨ 특수 효과

### 호버 & 인터랙션

#### 부드러운 전환 (Smooth Transitions)
```
기본 전환 시간: 300ms
느린 전환: 500ms (중요 요소)
빠른 전환: 150ms (빠른 피드백)

효과 함수: ease-in-out
```

#### 스크롤 애니메이션
```
원칙: 자연스럽고 부드러운 느낌
스타일: David Whyte처럼 명상적
효과:
  ├─ Fade In (요소 나타남)
  ├─ Slide Up (텍스트 올라옴)
  └─ Scale (미묘한 확대)

속도: 느리게 (1-2초)
```

### 그림자 (Shadows)

```
Elevation 0 (평면)
└─ 그림자 없음

Elevation 1 (살짝 위)

## 🎨 최종 정리

### Simon Sinek의 구성 (Structure)
```
히어로 (신달수 등장)
    ↓
철학 ("왜 이것을 만들었는가")
    ↓
HOW (신달수의 설명)
    ↓
증명 (실제 변화 사례)
    ↓
추천 (아재 소개)
    ↓
호출 (당신도 함께하세요)
    ↓
콘텐츠 (신달수의 목소리 계속)
```

### David Whyte의 디자인 (Aesthetics)
```
따뜻한 색상 (파란색 + 오렌지색 + 크림색)
    +
여유로운 여백 (호흡하는 공간)
    +
영상 중심 콘텐츠 (자동재생, 자연광)
    +
손글씨 감성 (인간미)
    +
부드러운 전환 (300ms, 명상적)
    =
신뢰 + 감동 + 연결
```
