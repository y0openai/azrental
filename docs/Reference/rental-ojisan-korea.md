# 렌탈 아재(おっさんレンタル) 서비스 가이드

## 📋 서비스 개요

렌탈 아재(おっさんレンタル, Ossan Rental)는 **일본의 독특한 인력 임대 서비스**로, 중년 남성(아저씨)을 시간 단위로 빌려서 다양한 활동을 함께할 수 있는 플랫폼입니다.

- **국가**: 일본
- **설립년도**: 2012년
- **운영 현황**: 10년 이상 지속적으로 운영 중(2024년 기준)
- **공식 웹사이트**: [rental-ojisan.com](https://rental-ojisan.com/)

## 🎯 비즈니스 모델

### 1. 기본 구조

| 요소 | 내용 |
|------|------|
| **상품** | 중년 남성(보통 40~60대) 인력 임대 |
| **시간당 요금** | 1,000엔(약 8,800~9,000원) |
| **등록 인력** | 약 100명 | 2025년 현재
| **임대 방식** | 온라인 예약 시스템 |
| **계약 단위** | 시간 단위 |

### 2. 서비스 이용 절차

1. **공식 웹사이트 방문**: rental-ojisan.com 접속
2. **인력 선택**: 프로필 확인 후 원하는 '아재' 선택
3. **카트에 추가**: 예약 예정자를 카트에 추가
4. **온라인 결제**: 인터넷을 통한 결제
5. **이메일 확인**: 선택된 인력으로부터 연락 대기
6. **만남 계획**: 구체적인 시간과 장소 협의

### 3. 수익 구조

- **시간당 수수료**: 1,000엔/시간
- **소비자 가격 책정**: 시간당 요금 기준
- **추가 비용**: 이동 거리, 특수 요청 등에 따른 추가 비용 가능
- **수익 배분**: 임대 인력과 플랫폼의 수익 분배 구조(상세 내용은 비공개)
