# 아재렌탈(AZ help me) 웹사이트 와이어프레임

**작성자**: 신달수
**작성일**: 2026년 1월
**목적**: IA를 기반으로 한 저충실도 레이아웃 설계 (Low-Fidelity Wireframes)

---

## 📋 와이어프레임 개요

**목표**: 각 페이지의 정보 구조, 콘텐츠 배치, CTA 배치를 정의하고 사용자 흐름을 시각화

**원칙**:
- 신달수의 철학을 모든 페이지에 시각화
- 모바일-퍼스트 반응형 디자인
- 신뢰 구축을 위한 리뷰/평점 우선 배치
- 명확한 행동 유도 (CTA)

**포함 페이지**:
1. 홈 (Home)
2. 아재 찾기 (Browse)
3. 아재 프로필 (Ajae Profile)
4. 예약 (Booking)
5. 신달수의 영역 (Founder Hub)
6. 커뮤니티 (Community)
7. 고객 대시보드 (Customer Dashboard)
8. 아재 대시보드 (Ajae Dashboard)

### 💡 개선 사항 (기존과의 주요 차이점)

**새로운 구조의 장점**:

| 항목 | 기존 | 개선된 버전 |
|------|------|-----------|
| **창업자 노출** | 히어로만 표시 | 전 페이지에서 계속 등장 (5번) |
| **설명 방식** | 추상적 가치 제시 | 신달수의 직접 음성으로 설명 |
| **신뢰 구축** | 후기 먼저 | 창업자 철학 먼저 → 후기 |
| **CTA 위치** | 3개 (분산) | 2개 (명확) + 추가 호출 |
| **스크롤 포인트** | 5개 섹션 | 7개 섹션 (더 풍부한 이야기) |
| **창업자 스토리** | 하단 링크 | 상단 + 중간 + 하단 (지속적) |

### 🎯 새 구조의 핵심 원칙

1. **"신달수가 주인공"**
   - 모든 섹션에 신달수의 목소리가 존재
   - 각 섹션이 "신달수가 이렇게 생각해요"로 시작

2. **"개인적 미션 전달"**
   - "영포티를 반박합니다" (직접적 가치관)
   - "왜 이것을 만들었는가" (동기 명확)
   - "당신의 경험을 나누세요" (호출)

3. **"사실성과 인간성"**
   - 추상적 가치 아닌 신달수의 구체적 이야기
   - 게임회사 PD 경력 → 문제 발견 → 해결책
   - 영상/비디오 메시지로 생동감

4. **"신달수 = 브랜드"**
   - 신달수의 철학 = 회사의 철학
   - 신달수가 할 말 = 마케팅 메시지
   - 신달수의 가치 = 서비스의 가치

### 📊 스크롤 깊이 및 체류 시간

- **섹션 1 (히어로)**: 창업자 첫 인상 (3초 → 5초 증가)
- **섹션 2 (철학)**: 깊이 있는 이해 (30초)
- **섹션 3 (HOW IT WORKS)**: 신달수 설명 (20초)
- **섹션 4 (STORIES)**: 실제 변화 증명 (45초)
- **섹션 5 (AJAES)**: 액션 (15초)
- **섹션 6 (BECOME)**: 신달수의 직접 호출 (40초)
- **섹션 7 (MESSAGE)**: 계속 알아보기 (30초)

**예상 총 체류 시간**: 2분 30초 (기존 1분 30초 대비 66% 증가)

## 🎨 색상 & 타이포그래피 가이드

### 색상 팔레트 (Design System에서 상세 정의)
- **Primary**: 신뢰감 있는 파란색 (비포트폴리오 예정)
- **Secondary**: 따뜻한 주황색 (활동감)

- **Neutral**: 회색 계열 (텍스트, 배경)

### 타이포그래피
- **Heading**: 큼, 굵음, 명확성
- **Body**: 읽기 쉬운 크기 (16px 이상 모바일)
- **Caption**: 보조 정보
