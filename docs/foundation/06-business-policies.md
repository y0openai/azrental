## 📌 PM 결정 기록 (Decision Log)

### 결정 1: 경량 에스크로 채택

**일시**: 2026-01-12

**배경**:
- 일본 Ossan Rental은 현금 직거래 (10년간 문제 없음)
- 한국 적용 검토 결과, 법적 리스크 발견

**검토된 옵션**:
| 옵션 | 내용 | 리스크 |
|------|------|--------|
| A. 현금 직거래 | 일본 모델 그대로 | 🔴 원천징수 누락 → 국세청 추징 |
| B. 경량 에스크로 | 결제만 플랫폼 경유 | ✅ 법적 안전 |
| C. 완전 에스크로 | 숨고 모델 | 🟡 운영 복잡 |

**결정**: **B. 경량 에스크로**

**결정 이유**:
```
1. 한국 세법상 플랫폼이 중개하면 원천징수 의무 발생
2. 현금 직거래 시 아조씨 소득 신고 누락 → 플랫폼에 추징
3. 경량 에스크로는 법적 리스크 제거 + 운영 단순성 유지
4. 소규모 시작(5명)에도 적용 가능
5. 일본 모델의 "아날로그 감성"은 서비스 품질로 유지
```

## 1. 결제 정책 ✅ 확정

### 1.1 결제 모델: 경량 에스크로

```
┌─────────────────────────────────────────────────────┐
│  경량 에스크로 플로우                                │
├─────────────────────────────────────────────────────┤
│                                                      │
│  1. 예약 확정 시                                     │
│     고객 → Toss 결제 → 플랫폼 보관                  │
│                                                      │
│  2. 서비스 완료 후 24시간 이내                       │
│     플랫폼 → 아조씨 계좌 자동 정산                   │
│     (수수료 10% + 원천징수 3.3% 차감)               │
│                                                      │
│  3. 분쟁 발생 시                                     │
│     플랫폼이 중재, 필요시 환불 처리                  │
│                                                      │
└─────────────────────────────────────────────────────┘
```

### 1.2 수수료

| 항목 | 값 | 비고 |
|------|-----|------|
| 플랫폼 수수료 | **10%** | 업계 표준 (숨고 10-20%) |
| 수수료 부담 | **아조씨** | 고객 결제액에서 차감 |
| 원천징수 | **3.3%** | 기타소득 (일시적 용역) |
| PG 수수료 | **포함** | 플랫폼 수수료에서 처리 |

**예시**:
```
고객 결제: 50,000원 (2시간 × 25,000원)
├─ 플랫폼 수수료: -5,000원 (10%)
├─ 원천징수: -1,650원 (3.3%)
└─ 아조씨 정산: 43,350원
```

### 1.3 정산 주기

| 항목 | 값 | 비고 |
|------|-----|------|
| 정산 시점 | 서비스 완료 후 **24시간** | 분쟁 제기 기간 |
| 정산 방식 | **자동 입금** | 등록된 계좌로 |
| 최소 정산액 | **없음** | 건별 즉시 정산 |

### 1.4 결제 수단

| 수단 | 지원 | 우선순위 |
|------|------|---------|
| Toss Payments | ✅ | 1순위 (메인) |

| 계좌이체 | ✅ | Toss 통해 지원 |

### 2.2 예약 확정 방식

**결정**: **아조씨 승인제**

```
고객 예약 요청 → 아조씨에게 알림 → 12시간 내 수락/거절
                                    ├─ 수락 → 예약 확정 → 결제 진행
                                    ├─ 거절 → 고객에게 알림
                                    └─ 무응답 → 자동 취소
```

**이유**: 아조씨 5명 소규모 운영, 각자 일정 관리 자율성 보장

## 4. 아조씨 승인 정책 ✅ 확정

### 4.1 가입 요건

| 항목 | 필수 | 이유 |
|------|------|------|
| 본인 인증 (휴대폰) | ✅ | 기본 신원 확인 |
| 신분증 사본 | ✅ | 법적 요건 (오프라인 보관) |
| 프로필 사진 | ✅ | 얼굴 필수 |
| 자기소개 | ✅ | 최소 100자 |
| 신용조회 동의 | ✅ | 신뢰성 검증 |
| 범죄경력 조회 | ⏳ Phase 2 | 법적 검토 후 |

### 4.2 심사 절차

**결정**: **서류 심사 + 영상 면접**

```
1단계: 서류 심사 (자동 + 수동)
├─ 신분증 확인
├─ 프로필 완성도
└─ 자기소개 내용

2단계: 영상 면접 (15분)
├─ 본인 확인
├─ 서비스 이해도
├─ 안전 규칙 교육
└─ 의도 파악

합격 기준: 엄격 (목표 합격률 2-5%)
```

**이유**: 소규모 운영이므로 품질 > 수량. 일본 Ossan Rental 합격률 2% 참고.

### 4.3 활동 유지 조건

| 항목 | 기준 | 미달 시 |
|------|------|--------|
| 최소 평점 | **4.0/5.0** | 경고 → 심사 |
| 응답률 | **80%** | 경고 |
| 신고 누적 | **2건** | 활동 정지 심사 |
| 휴면 기준 | **3개월** 무활동 | 휴면 전환 |

## 6. 신고/제재 정책 ✅ 확정

### 6.1 신고 사유

| 사유 | 심각도 | 처리 |
|------|--------|------|
| 지각 (15분+) | 🟡 낮음 | 경고 |
| 불친절 | 🟡 낮음 | 경고 |
| 서비스 범위 이탈 | 🟠 중간 | 경고 + 환불 |
| 연락처 요구 | 🟠 중간 | 경고 |
| 금전 요구 (외부) | 🔴 높음 | 즉시 정지 |
| 부적절 언행 | 🔴 높음 | 즉시 정지 |
| 성희롱/성추행 | 🔴 심각 | **영구 퇴출** |
| 폭언/폭행 | 🔴 심각 | **영구 퇴출** |

### 6.2 제재 단계

```
1단계: 경고 (기록)
2단계: 7일 활동 정지
3단계: 30일 활동 정지
4단계: 영구 퇴출

* 심각 위반(성희롱, 폭행)은 단계 무시 → 즉시 4단계
* 경고 3회 누적 → 2단계 자동 적용
```

### 6.3 신고 처리

| 항목 | 값 |
|------|-----|
| 처리 기한 | **48시간** 이내 |
| 처리 방식 | **관리자 검토** (자동화 없음) |
| 신고자 알림 | 결과 통보 (익명) |
| 이의 제기 | **1회** 가능 |

**이유**: 소규모 운영, 자동화보다 수동 검토가 정확함.
